# CINEMATIC UPI FINTECH EXPERIENCE

An immersive and interactive visualization of India’s revolutionary **UPI (Unified Payments Interface)** system — built using **p5.js, React, and TypeScript**.

This project transforms digital payments into a cinematic, real-time experience with animations, sound, and user interaction.

# Features

* Smooth animated transaction flows
* Interactive sound effects synced with actions
* User-triggered animations and controls
* Real-time visual feedback
* Clean and modern cinematic UI

# Concept

India’s UPI system has transformed digital payments globally.

This project visualizes:

* Peer-to-peer transactions
* Bank interactions
* Digital payment ecosystem flow
* UPI as a global inspiration

# Tech Stack

* React
* p5.js (Creative Coding)
* TypeScript
* Vite
* Web Audio API

# Project Structure

```bash
src/
├── App.tsx           # Main UI
├── main.tsx          # Entry point
├── sketch.ts         # p5.js animation logic
├── soundEngine.ts    # Audio interaction
```

# Getting Started

1. Clone the Repository

```bash
git clone https://github.com/sameerupadhyaytech/UPI-Ninjas_cc2026
cd YOUR-REPO
```

2. Install Dependencies

```bash
npm install
```

3. Run the Project

```bash
npm run dev
```

Open the application at:

```bash
http://localhost:5173/
```
